# pulsar-site

Landing page for Pulsar, a shared music broadcasting product by Relux Works. Static
site with English, Armenian, and Georgian locales, served via Cloudflare.

## Structure

- `index.html`: root redirect
- `en/`, `hy/`, `ka/`: localized pages
- `guide/`: user guide
- `legal/`: generated stable and immutable EN/RU policy/support pages; do not
  edit generated files by hand
- `_headers`: Cloudflare cache/security contract for legal routes

The legal bundle is generated by `coordinator/cmd/policy-site-build` in the
`relux-works/barycenter` commit pinned by `legal/deployment-manifest.json`.
Pull requests fail closed unless that exact source pack is owner-approved and
every generated byte matches.
